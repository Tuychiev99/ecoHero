# ecoHero
Django-ecommerce built on the Django Web Framework.
# Features Included
Custom Admin dashboard
Search Functionality
Shopping Cart
Order Management
Much more...

## Installation

*clone Repository & Install Packages**
```sh
git clone https://github.com/Tuychiev99/ecoHero-capstone-1
```
**2.Setup Virtualenv**
```sh
python3 -m venv venv
source venv/bin/activate
```
**3.Migrate & Start Server**
```sh
python3 manage.py migrate
python3 manage.py runserver
```
